Advanced Lane Finding

This project takes a deeper look into OpenCV functions by calibrating the vehicle camera, applying multiple thresholding techniques, and fitting higher order polynomials to the filtered lane lines. This can be used to measure the curvature of the road as well as identify where the vehicle is in the lane.

View Code

  • Technologies & Skills Used:
  • Python, OpenCV, NumPy, Git, Github
  • Computer Vision, ADAS, Self-Driving, Machine Learning

Finding Lane Lines

This project explores projecting left and right lane lines through the use of Canny edge detection, masking an area of interest, and highlighting identified lanes through Hough transformation. This allows us to project lines from points identified within the area of interest.

View Code

  • Technologies & Skills Used:
  • Python, OpenCV, NumPy, matplotlib
  • Computer Vision, ADAS, Self-Driving, Machine Learning
